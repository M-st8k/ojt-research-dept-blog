18/07/2023

Tue●Week 5

  • collapse expand

I’ve successfully made a script that corrects the image orientation, combines 2 pager abstracts into one, converts all jpegs into pdfs and compresses all these pdfs for smaller storage. For the orientation of the images I used a program called feh to view each of the images so that I could see their orientation, after I’ve determined that it is in landscape mode it Is just a matter of pushing a button to correct the orientation and save it. For the conversion of jpegs to pdfs. I used the “convert” command from imagemagick. It takes only 2 parameters, the input file and output filename and out goes the pdf version of the image. For the combination of 2 pager abstracts I used a Linux command called “find” that finds all directories containing files named “Abstract 1.jpg” and “Abstract 2.jpg” then executes a command within that directory. Within those directories I used the “pdftk” command from pdftricks to combine all pdf files globed using “Abstract*” This finds all files with “Abstract” in the name and combines it into one pdf output file. For the compression I used a program called ghostscript that can do a variety of things to a pdf file including compression, from 3Gb total size it went to just 1.6Gb which is more than 50% reduction. After all of these are done, I’ve uploaded It to google drive where it can be easily accessed later.
